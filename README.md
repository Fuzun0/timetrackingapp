# Time Tracking App

React Native ile geliştirilmiş zaman takip uygulaması.

## Öğrenci Bilgileri

**Hüseyin Furkan Uzun**  
**20235070309**

## Özellikler

- Timer ekleme, düzenleme ve silme
- Timer başlatma/durdurma
- Gerçek zamanlı sayaç güncellemesi
- Form ile timer yönetimi
- Expo Go ile mobil cihazlarda çalışma

## Kurulum

```bash
npm install --legacy-peer-deps
```

## Çalıştırma

```bash
npx expo start
```

Expo Go uygulaması ile QR kodu tarayarak uygulamayı mobil cihazınızda çalıştırabilirsiniz.

## Teknolojiler

- React Native
- Expo SDK 54
- React 19.1.0
